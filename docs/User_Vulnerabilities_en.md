# Vulnerabilities

## Overview

A comprehensive list of all vulnerabilities found across your projects. Use this centralized view to prioritize remediation efforts.

## Key Features

- **Global Search**: Find vulnerabilities by CVE ID or description.
- **Status Management**: Bulk update vulnerability status (e.g., mark as fixed).
- **Assignee Management**: Assign specific issues to team members.
