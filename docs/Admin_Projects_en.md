# Project Management

## Overview

Project Management allows administrators to oversee all projects within the organization. You can view project details, manage assignments, and track project health.

## Key Features

- **Project List**: comprehensive view of all projects.
- **Project Details**: Access specific configurations and scan histories.
- **Management**: Assign users to projects and configure project-specific settings.
