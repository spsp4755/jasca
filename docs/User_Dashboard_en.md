# Dashboard

## Overview

The User Dashboard provides developers and security leads with an immediate view of their project's security health. It highlights recent scans, critical vulnerabilities, and assigned tasks.

## Key Features

- **Security Grade**: Visual representation (A-F) of project security.
- **Trend Analysis**: Charts showing vulnerability trends over time.
- **My Tasks**: List of vulnerabilities assigned to the current user.
