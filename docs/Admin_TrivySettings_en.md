# Trivy Settings

## Overview

Configure the integration with Trivy, the security scanner. Manage server endpoints, authentication tokens, and scan preferences.

## Key Features

- **Server Configuration**: Set Trivy server URL and port.
- **Authentication**: Manage API tokens for secure connection.
- **Scan Options**: Configure default scan parameters.
