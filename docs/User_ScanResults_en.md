# Scan Results

## Overview

Detailed reports of security scans performed on the projects. Filter results by severity, status, or scanner type.

## Key Features

- **Result Filtering**: Focus on Critical or High issues.
- **Detailed View**: Inspect code snippets and remediation suggestions.
- **Status Update**: Mark issues as False Positive or Fixed.
